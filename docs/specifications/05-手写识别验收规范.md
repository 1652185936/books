# 墨学书架手写识别验收规范

| 项目 | 内容 |
|---|---|
| 文档编号 | MX-HWR-001 |
| 版本 | 1.1.0 |
| 状态 | 评审修订版 |
| 关联需求 | NFR-P-004、NFR-P-007、FR-F07 |
| 编写日期 | 2026-06-19 |

## 1. 验收目标

验证“用户点击确认答案后，识别服务返回的文本”是否满足：

- 整体空位精确识别准确率不低于 90%。
- P95 识别服务耗时不高于 1.5 秒。
- 服务失败时不产生错误判分，并保留原始笔迹。

本文评估识别能力本身，不把答案是否符合语法计入识别准确率。

## 2. 固定验收数据集

数据集 ID：`MX-HWR-EN-2000-v1`。

| 类别 | 数量 | 说明 |
|---|---:|---|
| 单个英文单词 | 700 | 常见初级英语词汇，长度 1..12 |
| 2 个词短语 | 500 | 如 `is drinking`、`did not` |
| 3–4 个词短语 | 500 | 如 `has already finished` |
| 缩写与撇号 | 150 | 如 `isn't`、`I'm working` |
| 数字或英数混合 | 150 | 如 `8 o'clock`、`Unit 3` |
| 合计 | 2000 | 每个样本对应一个可判分空位 |

采集要求：

- 至少 40 名书写者，每人 50 个样本。
- 左利手不少于 5 人。
- 每位书写者不能看到识别结果后重写同一样本。
- 每个短语只能指定一个人工标注真值。
- 训练、调参和验收数据集不得重叠；提供样本 SHA-256 清单。
- 样本保存原始 stroke 文件，不使用图片转写替代真实笔迹采集。

## 3. 书写设备与环境

基准设备类别：

- HarmonyOS 平板。
- 屏幕分辨率不低于 2560 × 1600。
- 运行内存不低于 8 GB。
- 使用系统兼容主动式手写笔。
- 应用画布宽度 240..360 vp，高度至少 88 vp。

正式验收必须记录：

```text
deviceModel
HarmonyOSVersion
recognitionProvider
modelVersion
networkMode(local/remote)
datasetId
appBuild
```

若识别服务为远程服务，网络基线为下行/上行均不低于 10 Mbps、往返时延不高于 80 ms；
离线模型不使用网络条件。

## 4. 文本标准化

对识别输出和 ground truth 使用同一标准化函数：

1. Unicode NFC。
2. 去首尾空白。
3. 连续空白压缩为一个空格。
4. 弯引号统一为直引号。
5. 英文字母转小写。
6. 不忽略单词内部标点，不纠正拼写，不补词。

## 5. 指标公式

### 5.1 空位精确识别准确率

```text
ExactBlankAccuracy =
  标准化后识别文本与 ground truth 完全相同的样本数
  / 所有已返回文本的样本数
```

### 5.2 端到端成功率

```text
EndToEndSuccessRate =
  标准化后完全相同的样本数
  / 全部 2000 个样本
```

服务超时、空文本、不可用和内部错误都计入端到端失败。

### 5.3 词错误率

```text
WER = (Substitution + Deletion + Insertion) / ground truth 总词数
```

WER 用于诊断，不替代 90% 的精确准确率门槛。

### 5.4 时延

从用户点击“确认答案”且请求离开应用服务层时开始计时，到收到最终识别结果或失败为止。
按 2000 个样本计算 P50、P95 和最大值。

## 6. 通过标准

必须同时满足：

| 指标 | 门槛 |
|---|---:|
| ExactBlankAccuracy | ≥ 90.0% |
| EndToEndSuccessRate | ≥ 88.0% |
| P95 时延 | ≤ 1.5 秒 |
| 单个请求硬超时 | 5 秒 |
| 因识别失败而产生的错误判分 | 0 |

数值按原始计数计算后保留两位小数，不先对分组结果取平均。

## 7. 执行与审计

- 由自动化脚本逐个回放 stroke 样本并保存原始输出。
- 结果文件必须包含每个样本的真值、识别值、标准化值、耗时和错误码。
- 汇总报告附数据集清单哈希、模型版本和设备信息。
- 任一门槛未满足即验收失败；更换模型或归一化算法后必须完整重跑。
- 验收集本身由两人抽样复核至少 10%；发现标注错误先修订数据集版本再重跑，不得手改结果。
